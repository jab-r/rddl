A Resource Directory Description Language (RDDL) specification for HTML 4.01

The nature of an HTML document is identified by the URI http://www.rddl.org/natures/html#v4.01. This document serves as a RDDL description of the HTML 4.01 Specification.

SGML definition

The formal SGML definition is contained in Section 19:

    Default DTD:

  • The nature of HTML validating to strict.dtd is http://www.rddl.org/natures/html#v4.01-strict.

    • Transitional DTD:

  • The nature of HTML validating to loose.dtd is http://www.rddl.org/natures/html#v4.01-transitional.

    • Frameset DTD:

  • The nature of HTML validating to frameset.dtd is http://www.rddl.org/natures/html#v4.01-frameset.
